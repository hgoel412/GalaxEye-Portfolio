# Dashboard Gallery

Each dashboard tells part of the GalaxEye story. Generated from 62 STK scenario files and 357K+ data points.

## 1. GalaxEye Constellation Overview
**File:** `01_GalaxEye_Constellation.jpg`

Hero dashboard showing:
- Walker constellation geometry (6, 12, 32 satellites)
- Ground track patterns
- Coverage heatmaps for India
- Key statistics (orbital period, coverage %, revisit time)

**Audience:** Executives, recruiters, system architects
